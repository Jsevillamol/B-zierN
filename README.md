B-zierN
=======

Python support to work with Bézier curves of grade n. It includes a class with methods such as affine transformation, restriction to a sub-interval, polarization and degree elevation.

Features
--------
+ Bézier curve evaluation (De Castlejau algorithm and explicit equation implementation)
+ Affine tranformation
+ Blossom
+ Restriction to a subinterval and curve split
+ Degree elevation
